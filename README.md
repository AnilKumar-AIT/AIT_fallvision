# FallVision

A React-based Sleep-Wake Disorders Dashboard for monitoring patient sleep patterns.

## Tech Stack
- React
- Recharts
- React Scripts (Create React App)

## Getting Started

### Install dependencies
npm install

### Run locally
npm start

### Build for production
npm run build
